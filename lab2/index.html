5. What does a variable scope mean

Ans: There are two types of scopes in programming:
  • Global scope
  • Local scope
Example:
  • Global scope:
    var name = 'Tayyab';
    console.log("My name is", name); Output: My name is Tayyab

  • Local scope:
    function handler() {
    var name = 'M.Tayyab';
    console.log("My name is", name);
    }
    handler()
    Output: My name is M.Tayyab
